    Setup

    I wanted to create a workflow for writing in public with ease.

    My requirements are: 1. Control the whole process from the command line 2. Write in markdown 3. A very fast site 4. No manual management over servers 5. Custom domain name

    Requirements 1-3 are easily satisfied by hugo. I also like the API of hugo, which I will go over in another post.

    Hugo also pretty simply supports rendering on github pages: https://gohugo.io/hosting-and-deployment/hosting-on-github/

    The workflow

    hugo new posts/intial-commit.md

    In a background terminal or tmux pane, start the hugo local server process:
    hugo server -D
    Now open http://localhost:1313 for a preview of the blog.

    I do my editing with vim, and one of the best things with Markdown is not having to flip back and forth between the browser and the text editor, except when checking image layout and the like.
